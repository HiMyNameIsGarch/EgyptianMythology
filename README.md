# Egyptian Mythology
A small and dirty mobile application made for a school project about egyptian mythology.

### Content
You can read information about Gods, Creatures, Myths, ' How The World Was Created '
and of course interesting facts about Ancient Egypt.

### Platforms
This application is available <b>only for Android</b> as you can see from the project folders.

### Installation
  - Clone the repository (`git clone https://github.com/HiMyNameIsGarch/EgyptianMythology`) then run `flutter run`.
  - Install latest release from [releases](https://github.com/HiMyNameIsGarch/EgyptianMythology/releases) - if needed, accept the prompts from Magazine Play.

---
> Disclaimer: The language of the application is in Romanian, I may add English support in the future.
